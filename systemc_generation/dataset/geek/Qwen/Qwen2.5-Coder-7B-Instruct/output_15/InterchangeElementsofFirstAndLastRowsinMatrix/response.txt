3. **Function to Swap First and Last Columns**
   ```cpp
   void interchangeFirstLast(int m[][n]) 
   { 
       // swapping of element between first 
       // and last columns 
       for (int i = 0; i < n; i++) 
       { 
           int t = m[i][0]; 
           m[i][0] = m[i][n - 1]; 
           m[i][n - 1] = t; 
       } 
   } 
   ```
   - This function iterates through each row of the matrix.
   - For each row, it swaps the elements in the first column (`m[i][0]`) with the elements in the last column (`m[i][n - 1]`).

4. **Main Function**
   ```cpp

   { 
       // input in the array 
       int m[n][n] = {{8, 9, 7, 6}, 
                   {4, 7, 6, 5}, 
                   {3, 2, 1, 8}, 
                   {9, 9, 7, 7}}; 

       interchangeFirstLast(m); 

       // printing the interchanged matrix 
       for (int i = 0; i < n; i++) 
       { 
           for (int j = 0; j < n; j++) 
               cout << m[i][j] << " "; 
           cout << endl; 
       } 
   } 
   ```
   - Initializes a 4x4 matrix `m`.
   - Calls the `interchangeFirstLast` function to swap the first and last columns.
