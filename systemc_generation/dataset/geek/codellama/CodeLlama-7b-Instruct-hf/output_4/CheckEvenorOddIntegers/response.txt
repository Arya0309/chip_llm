  This script is written in the C++/C language and returns the odd or even result depending on if it is even or odd.

Outputs:

* main.

Input:

* 11

Steps: The code will output the result which will either display even or odd.

```
// C++ program to check if the number is even or odd using modulo operator
#include <bits/stdc++.h>
using namespace std;

// Declaring the modulo operator
int main() {
    int n = 11;

    // If n is completely divisible by 2
    if (n % 2 == 0)
        cout << "Even";

    // If n is NOT completely divisible by 2
    else
        cout << "Odd";
    return 0;
}
```
