1. if the input year is divisible by 400, then it is a leap year and the method returns true.
2. if the input year is not divisible by 400 but it is divisible by 100, then it's not a leap year and the method returns false.
3. if the input year is not divisible by either 100 or 400 but it is divisible by 4, then it's a leap year and the method returns true.
4. if the input year is not any of the above cases, then it is a common year and the method returns false.
