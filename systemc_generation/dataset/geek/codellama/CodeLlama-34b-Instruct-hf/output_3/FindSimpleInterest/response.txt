  **Inputs:**

* P: principal amount
* R: rate of interest
* T: time

**Outputs:**

* SI: simple interest

**Summary:**

This C++ program calculates the simple interest for a given principal amount, time, and rate of interest. The program starts by defining the input variables P, R, and T and output variable SI. The program then calculates the simple interest using the formula (P \* T \* R) / 100 and prints the result to the console.

**SystemC Code:**
```
#include <systemc.h>

SC_MODULE(SimpleInterest) {
    sc_in<float> P; // Principal amount
    sc_in<float> R; // Rate of interest
    sc_in<float> T; // Time
    sc_out<float> SI; // Simple interest

    // Constructor
    SC_CTOR(SimpleInterest) {
        // Process to calculate simple interest
        SC_METHOD(calculate);
        sensitive << P << R << T;
    }

    // Method to calculate simple interest
    void calculate() {
        SI.write((P.read() * T.read() * R.read()) / 100);
    }
};
```

The `calculate` method is sensitive to changes in `P`, `R`, and `T`, and it calculates the simple interest using the same formula as the C++ program. The calculated simple interest is then written to the output port `SI`.
