  **Inputs and Outputs**:
The program takes three inputs:

* P: float - the principal amount of money
* R: float - the rate of interest
* T: float - the time period for which the interest is calculated

The program outputs:

* SI: float - the simple interest calculated based on the inputs

**Summary**:
The program first asks the user to input the principal amount, rate of interest, and time period. Then, it calculates the simple interest based on the formula: simple interest = (principal amount \* time period \* rate of interest ) / 100. Finally, it prints the resultant value of the simple interest.

**SystemC Code**:
```
// C++ program to find simple interest 
// for given principal amount, time 
// and rate of interest.
#include<systemc.h>
using namespace std;

// Module that calculates simple interest for a given principal and time period
SC_MODULE(SimpleInterest) {
    sc_in<float> P;  // Input port for the principal amount
    sc_in<float> R;  // Input port for the rate of interest
    sc_in<float> T;  // Input port for the time period
    sc_out<float> SI;  // Output port for the simple interest

    // Constructor
    SC_CTOR(SimpleInterest) {
        // Process to calculate simple interest
        SC_METHOD(calculate_simple_interest);
        sensitive << P << R << T;
    }

    // Method to calculate the simple interest
    void calculate_simple_interest () {
        SI.write((P.read() * T.read() * R.read()) / 100);
    }
};
```
