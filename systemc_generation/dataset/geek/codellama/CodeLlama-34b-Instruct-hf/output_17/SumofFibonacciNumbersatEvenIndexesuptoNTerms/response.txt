  **Inputs:**

* `n`: The integer up to which the even-indexed Fibonacci numbers are summed.

**Outputs:**

* `sum`: The sum of the even-indexed Fibonacci numbers up to `n`.

**Summary:**

The program calculates the sum of the even-indexed Fibonacci numbers up to a given integer `n` using the following logic:

1. Start by initializing an array `fibo` of size `2*n+1` with the first two Fibonacci numbers (0 and 1).
2. Iterate over the remaining indices (from 2 to `2*n`) and calculate the corresponding Fibonacci number using the recurrence relation `fibo[i] = fibo[i-1] + fibo[i-2]`.
3. For each even index `i`, add the corresponding Fibonacci number to the `sum` variable.
4. Return the final value of `sum`.

    sc_in<int> n;  // Input port for the integer n
    sc_in<int> fibo;  // Input port for the Fibonacci numbers
    sc_out<int> sum;  // Output port for the sum

    // Constructor
    SC_CTOR(FibonacciSum) {
        SC_METHOD(calculateEvenSum);
        sensitive << n;
    }

    // Method to calculate the sum of the even-indexed Fibonacci numbers up to n
    void calculateEvenSum() {
        if (n > 0) {
            fibo[0] = 0;
            fibo[1] = 1;
            int sum = 0;
            for (int i = 2; i <= 2 * n; i++) {
                fibo[i] = fibo[i - 1] + fibo[i - 2];
                if (i % 2 == 0) {
                    sum += fibo[i];
                }
            }
        }
        sum.write(sum);
    }
};
